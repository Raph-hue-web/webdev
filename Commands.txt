-cd folderName

-mkdir folderName

*Initialize Git
-Git init

*stage Git
-git add .

*Commit 
-git commit -m" message"

*Checking branches 
-git branch

*Creating a branch
-git branch branchName

*enter a branch
-git checkout branchName

*Merging Branches
-git merge branchName

*Deleting Branch 
-git branch -D branchName
